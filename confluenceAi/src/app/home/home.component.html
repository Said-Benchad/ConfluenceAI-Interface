Configurer l'authentification LDAP dans Confluence

Accédez à l'Administration de Confluence :

  • Connectez-vous en tant qu'administrateur.
  • Allez dans Administration > Gestion des utilisateurs > Sources d'annuaire.

Ajouter une source d'annuaire :

  • Cliquez sur Ajouter un annuaire et sélectionnez Annuaire LDAP.

Configurer les paramètres LDAP :

  • Serveur LDAP :
    • Adresse du serveur : ldap://votre-serveur-ldap:389
    • Base DN : dc=exemple,dc=com
  • Authentification :
    • Nom d'utilisateur : cn=admin,dc=exemple,dc=com
    • Mot de passe : votre-mot-de-passe
  • Mappage des groupes :
    • Groupe de base DN : ou=groupes,dc=exemple,dc=com
  • Mappage des utilisateurs :
    • Utilisateur de base DN : ou=utilisateurs,dc=exemple,dc=com

Tester et sauvegarder :

  • Cliquez sur Tester la connexion pour vérifier les paramètres.
  • Si le test est réussi, cliquez sur Enregistrer pour ajouter l'annuaire.

Pour des détails supplémentaires, vous pouvez consulter la page de configuration LDAP sur Confluence.
